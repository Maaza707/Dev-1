# 🚀 نظام إدارة الموارد البشرية (HR Management System)

نظام بسيط واحترافي مبني باستخدام تقنيات الويب الأساسية لإدارة شؤون الموظفين، الحضور، والإجازات. تم تصميم هذا المشروع كجزء من رحلة تعلم الـ **DevOps** وتطوير تطبيقات الويب.

---

## 🛠️ التقنيات المستخدمة (Tech Stack)
* **HTML5:** لهيكلة صفحات النظام.
* **CSS3:** لتصميم واجهة مستخدم داكنة (Dark Mode) وعصرية.
* **JavaScript (ES6):** للتعامل مع البيانات والعمليات التفاعلية.

---

## ✨ المميزات الرئيسية
- [x] **إدارة الموظفين:** إضافة وحذف بيانات الموظفين.
- [x] **تتبع الحضور:** تسجيل حالة الحضور والغياب يومياً.
- [x] **نظام الإجازات:** طلب وتوثيق الإجازات السنوية والمرضية.
- [x] **تقييم الأداء:** لوحة لمتابعة مستوى الموظفين.
- [x] **واجهة تفاعلية:** تصميم متجاوب (Responsive) ومريح للعين.

---

## 📸 نظرة على النظام
> الموقع يتكون من صفحة ترحيبية تشرح المشاكل التي يحلها النظام، تليها لوحة التحكم الإدارية.
